Forex & MNB árfolyam app

Ez az alkalmazás egy Java Spring Boot-ra épülő szerveroldali webapp, ami bemutatja a valós idejű devizaárfolyamok kezelését és a Forex műveleteket. A frontend a HTML5 UP – Dimension reszponzív sablont használja.

Fő funkciók

  • SOAP – Magyar Nemzeti Bank (MNB) árfolyamok lekérdezése dátumtartományra, Chart.js grafikonon.
  • Forex-account – Számlainformációk megjelenítése (egyenleg, deviza, margin, nyitott ügyletek száma).
  • Forex-AktÁr – Kiválasztott instrumentum aktuális ára.
  • Forex-HistÁr – Az utolsó 10 historikus ár, választható granularity mellett.
  • Forex-Nyit – Pozíció piaci áron nyitása (pozitív: Long, negatív: Short).
  • Forex-Poz – Nyitott pozíciók listája táblázatban (instrumentum, mennyiség, ár, trade ID).
  • Forex-Zár – Pozíció zárása trade ID alapján, piaci áron.
